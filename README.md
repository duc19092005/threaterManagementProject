# Đồ án CNPMNC

## Cách chạy 


**Bước 1 : Thêm vào backend/backend thêm file appsettings.Production.json nội dung file e sẽ gửi lại 

**Bước 2 : Chạy lệnh

```bash

docker compose up -d

```

**Bước 3 : Access vào cho từng services

```text

FE : http://localhost:3000

BE : http://localhost:5229/swagger/index.html

Database : (Địa chỉ IP) hoặc (localhost) hoặc (local),1433

```
